Introduction to HTML:

• Html is a hyper text markup language

• It is not a programming language

• Hyper text means text that link to another text

• markup language is a computer language used to structure, format, and present text and data by adding special tags or symbols around content.

• Html is used to build the static structure of web page

• With the help of tags we build the structure of web page

Ex: build the house, means with the help of cement and bricks we can build the static structure of home
